# 🗓️ Day 7 – Shopping List CLI

A simple **command-line shopping list app** in Python.  
You can **add, remove, view, clear, and checkout** your shopping items.  

### Features
- 📋 View shopping list  
- ➕ Add item  
- ❌ Remove item  
- 🧹 Clear list  
- 💳 Checkout  

### Run the program
```bash
python main.py
