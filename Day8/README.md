# 🗓️ Day 8 – Contact Book App

A Python program to manage your contacts easily.  
You can **add, view, search, edit, and delete contacts** directly from the terminal.

### Features
- ➕ Add a new contact  
- 📋 View all contacts  
- 🔍 Search for a contact  
- ✏️ Edit existing contact  
- ❌ Delete contact  
- 🚪 Exit the program  

Run with:
```bash
python main.py
